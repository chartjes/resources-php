Level 4 Sequence Exercise
-----------------------
Create a function which returns the xp for a given level. Use the following sequence for the algorithm.
||xp||level||
|100|1|
|300|2|
|600|3|
|1000|4|
|1500|5|

##Rules:
* the first param of the function may be the level
